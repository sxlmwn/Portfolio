# Salman Younus — Portfolio

Live Demo: [https://sxlmwn.github.io/Portfolio/](https://sxlmwn.github.io/Portfolio/)

A modern, highly interactive portfolio showcasing my work as an AI, DevOps, and Security Engineer. Built with a focus on deep technical tracks, clean design, and smooth user experiences.

## Features

- **Modern Tech Stack**: React, Vite, TypeScript
- **UI & Styling**: Tailwind CSS, Shadcn UI, Framer Motion for smooth animations
- **Responsive Design**: Fully mobile responsive and interactive layout
- **Three Core Tracks**: Highlights AI / ML Engineering, Cloud-Native DevOps, and Offensive Cybersecurity.

## Local Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/sxlmwn/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

This project is configured to be deployed on GitHub Pages. To deploy updates:
```bash
npm run deploy
```
